# 289 Scoutz Linkz App

This is a Flutter web application for the 289 Scoutz Linkz project.

## 🚀 Getting Started

Follow these steps to run the app locally in your browser.

### 1. Open the Project

* Open the project in your IDE (e.g., VS Code).
* Open a terminal inside your IDE.

### 2. Navigate to the Project Folder

Make sure you are at the project directory:

```
289_scoutlinkz_app
```

---

### 3. Run the App in Your Browser

Run the following command:

```
flutter run -d chrome
```

This will launch the app in Google Chrome. (give it a sec)

---

### ⚡ Hot Reload / Restart

While the app is running:

* Press **`r` (lowercase)** → Hot reload (quick UI updates)
* Press **`R` (uppercase)** → Full restart of the app

---

## 📝 Notes

* Make sure Flutter is installed and set up properly.
* Ensure Chrome is installed on your machine.
* If the command doesn’t work, try running:

  ```
  flutter doctor
  ```

  to check your setup.

---

That’s it — you should be able to run and test the app right away 👍
